# Contributing to ContextWeaver

## Label Taxonomy

The following labels are used to categorize issues:

* `area/*`: context, routing, gateway, adapters, docs, benchmarks
* `complexity/*`: already exists
* `good first issue`
* `help wanted`
* `agent-friendly`
* Type labels:
	+ `enhancement`
	+ `bug`
	+ `documentation`
	+ `integration`
	+ `performance`
	+ `security`
	+ `testing`
	+ `developer-experience`
